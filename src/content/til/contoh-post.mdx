Hari ini struktur portfolio mulai dirapikan:

- Root layout menampung font, metadata, Navbar, dan Footer.
- Landing page dipecah menjadi section kecil agar mudah dipoles.
- Route terpisah disiapkan untuk projects, TIL, gallery, life-log, dev-stats, dan guestbook.
- API route Wakatime dan Supabase dibuat aman saat env var belum diisi.

Langkah berikutnya adalah mengganti starter copy dengan konten personal yang lebih akurat.
